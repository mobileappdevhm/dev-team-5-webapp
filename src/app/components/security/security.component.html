Auf der Webseite der CiE App werden ausschließlich Daten von jedem User benutzt, die über einen NINE-Account verfügen


Um einen NINE-Account anzulegen bitte den folgenden Link klicken.

Cookies

Auf der CiE Webseite werden keine Cookies verwendet, sondern ausschließlich Daten, die über den Studenten von Relevanz sind (Profil, User Name, etc.)
